## Changes in this version (1.3.3)

* Removed the pre-generated src/SimulationFunctions.cpp from the package
  sources; it is now regenerated by rstantools at install time via
  configure/configure.win. The previous checked-in copy was produced by an
  older stanc and would not compile against the Stan C++ headers in the
  forthcoming rstan 2.39.0. Contributed by Andrew Johnson (@andrjohns).
* Replaced non-ASCII em dashes in inst/stan/SimulationFunctions.stan comments,
  which caused stanc to fail under the rstan backend.
* Added "gamma1" support to GenerateMDCEVData() and a corresponding note in
  print(summary()).
* Exported and documented the low-level compiled simulation functions
  (DrawError_rng, MarshallianDemand, HicksianDemand, ComputeUtilJ,
  CalcWTP_rng, CalcMarshallianDemand_rng) and the RNG helpers
  rmdcev_get_rng / rmdcev_get_stream.
* Fixed GenerateMDCEVData() ignoring R's RNG state; set.seed() now controls
  reproducibility as expected.
* Relaxed the precision of several test snapshots that were sensitive to
  platform-specific optimizer output.
